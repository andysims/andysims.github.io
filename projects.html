  • Automation (I will continue to add more projects)
    • Export Census KMZs
      • Python script that: Reads and compares nationwide census blocks (Postgres) to client's eligible blocks (CSV). Generates directories for each state. Grabs county fips codes to generate names of KMZ files (ex: CA_LosAngeles.kmz). Using ArcPy, I setup the MXD, set definition queries hitting tables in Postgres, which then spits out a KMZ file for all counties in the U.S. where eligible.
    • Export Study Area KMZs
      • Python script that: Reads incoming CSV file of companies and their study areas to generate KMZ files. Use ArcPy to set definition query, hit Postgres tables, then exports a KMZ file for each company.
        • Study Area Check
          • Runs through directory to ensure each company successfully had a KMZ exported.
